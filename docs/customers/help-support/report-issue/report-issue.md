# Report an Issue

Have you encountered a problem while using the platform?  
You can report issues directly to our support team.

---

## How to Report a Problem

1. Go to the **Komashi Help Center**.  
2. Select the **Support** and **Chat** option.  
3. Describe the issue you are experiencing as **clearly and precisely** as possible.  
4. If available, **attach a screenshot** or any relevant information that helps illustrate the problem.

Providing detailed information allows our team to identify and resolve the issue faster.

---

Our support colleagues will review your report and assist you as soon as possible.
