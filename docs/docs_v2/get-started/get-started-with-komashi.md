## Complete setup to start selling

1. **Create your first service**  
   Add the service you want to sell in Komashi.

2. **Verify payment method**  
   Confirm your payment details so you can receive payouts.

3. **Set up payment**  
   Enable payment methods so customers can pay for your service.

**Complete these steps to start selling in Komashi.**
