# User and Permissions

You can add users to your provider account to help you manage your services.

---

## 1. Access Users and permissions 

Navigate to:

**Settings → Users and permissions**

Here you can view and manage all existing users connected to your account.

---

## 2. Add a New User

Click the **Add User** button to connect a new user.

You will be presented with a list of available users.

---

## 3. Select a User type

Choose the user type you want to add from the list.
admin, operator

### 👉 Click **Send Invite**

This will send an invitation to the user to join your provider account.

---

Your user setup is now complete.
