## Billing profile

A **billing profile** is a separate billing and business profile inside your account.

You can create **multiple billing profiles within a single account**. Each billing profile can have its **own services, products, and billing details**.

This allows you to **manage different businesses, projects, or product groups separately**.

For example, you might use different billing profiles if:
- you run **multiple businesses**
- you sell **different types of services**
- you want **separate billing information** for different activities

Each billing profile keeps its **services, customers, and financial data organized**, so you don't have to mix everything together in one place.
