# Customers

This guide explains where and how you can view and manage your customers as a provider.

---

## Accessing Customers

1. Log in to the **Komashi Control Center**.
2. Select the **provider** whose customers you want to view.
3. From the left-side navigation, click **Customers**.

Here you will see a list of **all customers** associated with the selected provider.

---

## Viewing Customer Details

If you want to view a specific customer:

1. Select the customer from the list.

You will then be able to see detailed information, including:
- Basic customer data
- Active and past subscriptions
- Issued invoices
- External or custom data associated with the customer
