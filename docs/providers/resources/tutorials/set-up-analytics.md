
# Set Up Analytics

Configure provider-specific Google Analytics and Microsoft Clarity tracking IDs.

---

## Google Analytics

Track visitor activity with a Google Analytics Measurement ID  
(for example: **G-XXXXXXXXXX**).

Click **Edit Tracking** to add or update your Google Analytics Measurement ID.

---

## Microsoft Clarity

Use a Clarity Project ID to enable session recording, heatmaps, and behavior insights.

Click **Edit Tracking** to add or update your Microsoft Clarity Project ID.

---

Once tracking IDs are saved, analytics data will begin collecting automatically.
